Renesas R-Car S4 (R8A779F0)
Device Tree Bindings

本ドキュメントは、Renesas R-Car S4 (R8A779F0) 上で動作する MFIS IPCC Mailbox および R-Car Remoteproc ドライバに必要な Device Tree バインディングを定義します。

対応ドライバは oot/mailbox/rcar-ipcc.ko および oot/remoteproc/rcar_rproc.ko として kernel 6.12 向け Out-of-Tree モジュールで提供されます。

アーキテクチャ

Application Processor (CA55) │ ├──> rcar_rproc.ko (remoteproc framework) │ │ resets ──> CPG (reset #222) │ │ mboxes ──> rcar-ipcc.ko (MFIS IPCC mailbox) │ │ │ interrupts ──> GIC SPI 320 (tx) │ │ │ ──> GIC SPI 322 (rx) │ │ syscfg ──> mfisarembr (syscon) (resource table addr) │ │ memory ──> cr52_ram @ 0xe2100000 (firmware) │ ──> vdev0vring0 @ 0x42000000 (VirtIO TX ring) │ ──> vdev0vring1 @ 0x42001000 (VirtIO RX ring) │ ──> vdev0buffer @ 0x42002000 (VirtIO buffer) │ └──> RPMSG / VirtIO <══════> CR52 (Cortex-R52)

メモリマップ

ラベル ベースアドレス サイズ 用途
mfis 0xe626000044 KB (0xB000)MFIS IPCC レジスタ
mfisarembr 0xe626946032 B (0x20) リソーステーブルアドレス格納レジスタ

Mailbox — renesas,rcar-ipcc NEW

MFIS IPCC (Inter-Processor Communication Controller) は、 Application Processor (CA55) とリアルタイムプロセッサ間の双方向メッセージング ハードウェアです。R-Car Gen3 / Gen4 に搭載される MFIS ブロックを使用します。

バインディングファイル: Documentation/devicetree/bindings/mailbox/renesas,rcar-ipcc.yaml

compatible

値対象 SoC
renesas,rcar-gen3-ipccR-Car Gen3 (H3, M3-W, M3-N など)
renesas,rcar-gen4-ipccR-Car Gen4 / S4 (R8A779F0 など)

プロパティ一覧

プロパティ型要否説明
compatiblestring 必須 renesas,rcar-gen3-ipcc または renesas,rcar-gen4-ipcc
regphandle 必須 MFIS レジスタ領域 (1 エントリ)
interruptsarray 必須 2 本: [0] TX 完了割込み, [1] RX 受信割込み
interrupt-namesstring-array 必須 "tx", "rx" の順
clocksphandle 必須 MFIS 動作クロック (通常 mfis_clk 固定クロック)
power-domainsphandle 必須 電源ドメイン (通常 R8A779F0_PD_ALWAYS_ON)
#mbox-cellsu32 必須 固定値 1。チャネルインデックス (0=TX, 1=RX) を指定する
#address-cellsu32 省略可 固定値 1
#size-cellsu32 省略可 固定値 0
チャネルインデックスについて #mbox-cells = <1> のため、mboxes プロパティで <&mfis 0> が TX チャネル、<&mfis 1> が RX チャネルを指します。

DTS 記述例 (R-Car S4)

/* 固定クロック — MFIS ブロック用 */
mfis_clk: fixed-clock {
    compatible = "fixed-clock";
    #clock-cells = <1>;
    clock-frequency = <1000000>;
    clock-output-names = "mfis";
};

/* MFIS IPCC メールボックス */
mfis: mailbox@e6260000 {
    compatible = "renesas,rcar-gen4-ipcc";
    reg = <0 0xe6260000 0 0xB000>;
    interrupts =
        <GIC_SPI 320 IRQ_TYPE_LEVEL_HIGH>,  /* tx */
        <GIC_SPI 322 IRQ_TYPE_LEVEL_HIGH>;  /* rx */
    interrupt-names = "tx", "rx";
    clocks = <&mfis_clk 0>;
    power-domains = <&sysc R8A779F0_PD_ALWAYS_ON>;
    #mbox-cells = <1>;
};

Remoteproc — renesas,rcar-rproc UPDATED

rcar-rproc は Renesas R-Car Gen3/Gen4 のリアルタイムプロセッサ (CR7, CR52) をロード・制御する remoteproc ドライバ向けバインディングです。 RPMSG/VirtIO 通信には MFIS IPCC メールボックスとの連携が必要です。

バインディングファイル: Documentation/devicetree/bindings/remoteproc/renesas,rcar-rproc.yaml

compatible

値プロセッサ世代備考
renesas,rcar-cr7 Cortex-R7Gen3 Linux が直接ファームウェアをロード。PA==DA (アドレス変換なし)
renesas,rcar-cr7_optee Cortex-R7Gen3 OP-TEE 経由でセキュアファームウェアをロード
renesas,rcar-cr52 Cortex-R52Gen4 / S4 オフセットベースのアドレス変換あり (DA = PA - device_mem_base)

プロパティ一覧

プロパティ型要否説明
compatiblestring 必須 上記 compatible 値のいずれか
resetsphandle 必須 CPG リセット制御 (1 エントリ)。例: <&cpg 222>
power-domainsphandle 必須 電源ドメイン (1 エントリ)
memory-regionphandle-array 必須 予約メモリ領域リスト (1 個以上)。
先頭エントリ: ファームウェア/RAM カーブアウト領域。
RPMSG 使用時は続けて VirtIO vring / バッファ領域を指定。
mboxesphandle-array 省略可 IPCC メールボックスチャネル 2 本 (TX, RX)。
RPMSG/VirtIO 通信を使用する場合は必要。
例: <&mfis 0>, <&mfis 1>
mbox-namesstring-array 省略可 mboxes に対応する名前。"tx", "rx" の順で指定
rcar,syscfg-rsc-tblphandle-array 省略可 リモートプロセッサがブート後にリソーステーブルのアドレスを書き込む syscon レジスタへの参照。
形式: <&syscon_node offset mask>
本プロパティが存在する場合、ドライバは DETACHED モードで起動し、 すでに動作中のリモートプロセッサへのアタッチが可能になる。
statusstring 省略可 SoC DTSI では "disabled"、ボード DTS で "okay" に設定する
CR52 アドレス変換について CR52 (Gen4) は memory-region の先頭エントリ (cr52_ram) の物理アドレスをデバイスアドレス空間の原点 (0x0) に マップします。ドライバは PA と DA の変換を自動的に処理します。
rcar,syscfg-rsc-tbl の使用条件 本プロパティは mfisarembr syscon ノードへの参照です。 リモートプロセッサが既に起動済みで、resource table アドレスがレジスタに 書き込まれている場合にのみ有効です。未起動時はドライバが警告を出して 通常モードで継続します。

DTS 記述例 — CR7 (Gen3)

/* reserved-memory ノード内 */
cr7_ram: cr7-ram@40040000 {
    no-map;
    reg = <0x0 0x40040000 0x0 0x1fc0000>;
};

/* CR7 リモートプロセッサ */
cr7_rproc: cr7 {
    compatible = "renesas,rcar-cr7";
    memory-region = <&cr7_ram>;
    power-domains = <&sysc R8A7795_PD_CR7>;
    resets = <&cpg 222>;
    status = "disabled";
};

DTS 記述例 — CR52 (Gen4 / R-Car S4)

/* SoC DTSI 側 (r8a779f0.dtsi) */
cr52_rproc: cr52@e2100000 {
    compatible = "renesas,rcar-cr52";
    power-domains = <&sysc R8A779F0_PD_ALWAYS_ON>;
    resets = <&cpg 222>;
    status = "disabled";
};

/* ──────────────────────────────────────────────── */

/* ボード DTS 側 (r8a779f0-yourboard.dts) */
reserved-memory {
    #address-cells = <2>;
    #size-cells = <2>;
    ranges;

    cr52_ram: cr52-ram@e2100000 {
        no-map;
        reg = <0x0 0xe2100000 0x0 0x200000>;    /* 2 MB */
    };
    vdev0vring0: vdev0-vring0@42000000 {
        no-map;
        reg = <0x0 0x42000000 0x0 0x1000>;     /* 4 KB */
    };
    vdev0vring1: vdev0-vring1@42001000 {
        no-map;
        reg = <0x0 0x42001000 0x0 0x1000>;     /* 4 KB */
    };
    vdev0buffer: vdev0-buffer@42002000 {
        no-map;
        reg = <0x0 0x42002000 0x0 0x4000>;     /* 16 KB */
    };
};

mfisarembr: syscon@e6269460 {
    compatible = "simple-bus", "syscon", "simple-mfd";
    reg = <0 0xe6269460 0 0x20>;
};

/* CR52 有効化 + RPMSG 設定 */
&cr52_rproc {
    memory-region = <&cr52_ram>,
                    <&vdev0vring0>,
                    <&vdev0vring1>,
                    <&vdev0buffer>;
    mboxes      = <&mfis 0>, <&mfis 1>;
    mbox-names  = "tx", "rx";
    rcar,syscfg-rsc-tbl = <&mfisarembr 0x0 0xFFFFFFFF>;
    status = "okay";
};

完全な DTS — SoC DTSI に追加するノード

以下を r8a779f0.dtsi の / ルートノード内に追加してください。 

/* MFIS 固定クロック */
mfis_clk: fixed-clock {
    compatible = "fixed-clock";
    #clock-cells = <1>;
    clock-frequency = <1000000>;
    clock-output-names = "mfis";
};

/* CR52 リモートプロセッサ (ベース定義) */
cr52_rproc: cr52@e2100000 {
    compatible = "renesas,rcar-cr52";
    power-domains = <&sysc R8A779F0_PD_ALWAYS_ON>;
    resets = <&cpg 222>;
    status = "disabled";
};

/* MFIS IPCC メールボックス */
mfis: mailbox@e6260000 {
    compatible = "renesas,rcar-gen4-ipcc";
    #mbox-cells = <1>;
    #address-cells = <1>;
    #size-cells = <0>;
    reg = <0 0xe6260000 0 0xB000>;
    interrupts =
        <GIC_SPI 320 IRQ_TYPE_LEVEL_HIGH>,
        <GIC_SPI 322 IRQ_TYPE_LEVEL_HIGH>;
    interrupt-names = "tx", "rx";
    clocks = <&mfis_clk 0>;
    power-domains = <&sysc R8A779F0_PD_ALWAYS_ON>;
};

/* MFIS レジスタブロック (リソーステーブルアドレス読み出し用) */
mfisarembr: mfisarembr@e6269460 {
    compatible = "simple-bus", "syscon", "simple-mfd";
    reg = <0 0xe6269460 0 0x20>;
};

完全な DTS — ボード DTS に追加するノード

以下を r8a779f0-yourboard.dts に追加してください。 

/* 予約メモリ領域 */
reserved-memory {
    #address-cells = <2>;
    #size-cells = <2>;
    ranges;

    cr52_ram: cr52-ram@e2100000 {
        no-map;
        reg = <0x0 0xe2100000 0x0 0x200000>;
    };
    vdev0vring0: vdev0-vring0@42000000 {
        no-map;
        reg = <0x0 0x42000000 0x0 0x1000>;
    };
    vdev0vring1: vdev0-vring1@42001000 {
        no-map;
        reg = <0x0 0x42001000 0x0 0x1000>;
    };
    vdev0buffer: vdev0-buffer@42002000 {
        no-map;
        reg = <0x0 0x42002000 0x0 0x4000>;
    };
};

/* CR52 有効化 */
&cr52_rproc {
    memory-region = <&cr52_ram>,
                    <&vdev0vring0>,
                    <&vdev0vring1>,
                    <&vdev0buffer>;
    mboxes      = <&mfis 0>, <&mfis 1>;
    mbox-names  = "tx", "rx";
    rcar,syscfg-rsc-tbl = <&mfisarembr 0x0 0xFFFFFFFF>;
    status = "okay";
};
